Comment tout faire marcher:

* Créer un compte Twitter
* Créer une application Twitter (avec les 4 ids nécessaires pour le fichier credentials.py)

Sur la VM:
* Installer docker:
   $ sudo apt-get update
   $ sudo apt-get install \
       apt-transport-https \
       ca-certificates \
       curl \
       software-properties-common
   $ curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo apt-key add -
   $ sudo add-apt-repository \
      "deb [arch=amd64] https://download.docker.com/linux/ubuntu \
      $(lsb_release -cs) \
      stable"
   $ sudo apt-get install docker-ce
* clone git
* Run docker :
  $ sudo docker run -p 5601:5601 -p 9200:9200 -p 5044:5044 -it --name elk sebp/elk (lance ELK dans un docker)
* Remplir le fichier src/credentials.py avec les 4 IDs
* src/twitterbot.py remplie la base
